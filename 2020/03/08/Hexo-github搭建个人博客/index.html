Hexo+github搭建个人博客
发表于 2020-03-08| 更新于 2020-03-09|hexo
阅读量:

Hexo:#

Hexo 是一款基于Node.js、快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。
####官网 https://hexo.io/zh-cn/docs/index.html

第一步：下载安装Git

  1. https://gitforwindows.org/

  2. 安装步骤:
    傻瓜式安装，只需一直下一步即可，然后选择一个安装地址

    第二步：下载安装node.js

  3. https://nodejs.org/en/

  4. 进行安装node.js和安装git一样一路默认就行

第三步：安装hexo

  1. 利用npm命令进行安装，在任意位置右击选择Git Bash Here
  2. 输入命令：npm install -g hexo

    第四步：初始化Hexo

  3. 创建文件夹（这个文件夹是博客的文件夹名字自定义）
  4. 在博客文件夹根目录下，右键运行git bash 输入命令：hexo init
  5. 这个命令输入完成后时间会有点长
  6. 等初始化完成后，在博客文件夹中会出现一些文件
  7. 在_config.yml进行配置，在config中配置theme为主题
    找到deploye字段，在下面加上repo为你在github创建的仓库地址
    branch字段为分支
  8. 进行完配置后再git中输入命令:hexo g 生成静态文件
  9. 输入hexo s 进行本地部署
  10. 本地部署完成后再浏览器中输入：loclhost:4000即可访问本地博客

    部署到GitHub上

  11. 申请github账号
  12. 选择new repository
  13. 然后点击 create repository
  15. 安装hexo-deployer-git自动部署发布工具
    1. npm install hexo-deployer-git –save
  16. 发布到Github
    1. 输入命令：hexo d
  17. 发布成功后访问你仓库的地址就可以了
